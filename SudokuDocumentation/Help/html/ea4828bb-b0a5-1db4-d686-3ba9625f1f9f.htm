Remarks
A possible solution for an unsolved cell. Each candidate represents a digit. Solving a sudoku puzzle is mainly done by elimination of candidates. When a cell contains a digit, the remaining values are no longer considered candidates for that cell. In addition, all peers of that cell lose their candidates for that digit, because each house can only contain one instance of each digit. @see RemovePossibleDigit(int candidate, SudokuResult child)
